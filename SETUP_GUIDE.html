üí° How the Spacer Works:

The .spacer { flex: 1 1 auto; } rule creates a flexible element that:

  • flex-grow: 1 - Grows to fill available space
  • flex-shrink: 1 - Can shrink if needed
  • flex-basis: auto - Uses automatic width calculation

This pushes everything after it (the nav links) to the far right of the toolbar.

TypeScript - projects.component.ts
import { Component, OnInit } from '@angular/core';
import { CommonModule } from '@angular/common';
import { MatCardModule } from '@angular/material/card';

interface Project {
  id: number;
  title: string;
  subtitle: string;
  description: string;
  technologies: string[];
  category: string;
  year: string;
  backgroundImage: string;
}

@Component({
  selector: 'app-projects',
  standalone: true,
  imports: [CommonModule, MatCardModule],
  templateUrl: './projects.component.html',
  styleUrl: './projects.component.scss'
})
export class ProjectsComponent implements OnInit {
  projects: Project[] = [];
  hoveredCardId: number | null = null;

  private projectsData: Project[] = [
    {
      id: 1,
      title: 'Drone Sim and Radio Controller Testing',
      subtitle: 'Beginning of Drone Studies',
      description: 'Journey into drone technology...',
      technologies: ['Liftoff', 'Radiomaster TX16S MAX II'],
      category: 'Personal',
      year: '2025',
      backgroundImage: 'https://images.unsplash.com/...'
    },
    // Add more projects...
  ];

  ngOnInit(): void {
    this.projects = this.projectsData;
  }

  onCardHover(cardId: number): void {
    this.hoveredCardId = cardId;
  }

  onCardLeave(): void {
    this.hoveredCardId = null;
  }
}
‚úÖ Component Features:
  • Project Interface - Defines the data structure
  • projectsData Array - Contains all project information
  • hoveredCardId State - Tracks which card is hovered
  • Event Handlers - onCardHover() and onCardLeave()
  • Standalone - Uses modern Angular standalone API

2.3Create Projects Component HTML

File: src/app/projects/projects.component.html

HTML - projects.component.html
<div class="projects-container">
  <h1>My Projects</h1>
  <div class="cards-grid">
    <mat-card 
      *ngFor="let project of projects" 
      class="project-card"
      (mouseenter)="onCardHover(project.id)"
      (mouseleave)="onCardLeave()"
    >
      <div class="card-content" [class.hovered]="hoveredCardId === project.id">
        <!-- Background Image (shows when not hovered) -->
        <div class="card-image" *ngIf="hoveredCardId !== project.id">
          <img [src]="project.backgroundImage" [alt]="project.title">
        </div>

        <!-- Project Data (shows when hovered) -->
        <div class="card-data" *ngIf="hoveredCardId === project.id">
          <h2>{{ project.title }}</h2>
          <p class="subtitle">{{ project.subtitle }}</p>
          <p class="description">{{ project.description }}</p>
          <div class="technologies">
            <span *ngFor="let tech of project.technologies" class="tech-tag">
              {{ tech }}
            </span>
          </div>
        </div>
      </div>
    </mat-card>
  </div>
</div>
‚úÖ Template Features:
  • *ngFor - Loops through project array
  • Mouse Events - (mouseenter) and (mouseleave)
  • Conditional Rendering - *ngIf for image/data views
  • Data Binding - {{ }} for dynamic content

  {path: 'about', component: AboutComponent, title: 'About'},
  {path: 'contact', component: ContactComponent, title: 'Contact Us'},
  {path: 'projects', component: ProjectsComponent, title: 'Projects'},
  {path: '', redirectTo: 'home', pathMatch: 'full'},
  {path: '**', component: PageNotFoundComponent}
];

‚ùì Troubleshooting

Issue: "Cannot find module '@angular/material/card'"

Solution:

ng add @angular/material
npm install

Issue: Projects page shows blank

Solution:

  • Check browser console for errors (F12)
  • Verify ProjectsComponent is imported in app.routes.ts
  • Ensure project data is populated in projectsData array

Issue: Navigation links not on right side

Solution:

  • Verify .spacer { flex: 1 1 auto; } is in navigation.component.scss
  • Check <span class="spacer"></span> is in HTML
  • Ensure nav { justify-content: flex-end; } is applied

Issue: Hover effect not working

Solution:

  • Check mouse event handlers are bound correctly
  • Verify hoveredCardId state management
  • Test in different browser if Chrome has issues
